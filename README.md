# Fake Personal Data Generator

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Node](https://img.shields.io/badge/node-20%2B-green.svg)
![React](https://img.shields.io/badge/react-18.3.1-61dafb.svg)

A simple web application to generate fake personal data for testing and development purposes.

## Features

- Swiss IBAN number generator
- Clean, modern UI built with React

## Tech Stack

- HTML
- CSS
- JavaScript
- React.js
- Vite (build tool)

## Getting Started

### Prerequisites

- Node.js 20 or higher
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/jonathanduss/FakePersonalData.git
cd FakePersonalData
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The build output will be in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```

## Usage

Simply click the "Generate IBAN" button to generate Swiss IBAN numbers.
